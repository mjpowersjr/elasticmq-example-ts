# elasticmq-example-ts
Simple example of using elasticmq in a ts project.

## Requirements
* Node v14+
* `yarn` or `npm`
* Docker (for elasticmq)

## Quickstart
```sh
# install deps
yarn install

# start elasticmq instance
yarn start-elasticmq

# start demo (in a new terminal, while elasticmq is running)
yarn start-demo
```
